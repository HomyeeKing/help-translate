# 根据浏览器窗口适配页面大小 🚀🚀

> 仅限谷歌浏览器

### 使用方法：

把文件克隆到本地

```git
 git clone git@github.com:HomyeeKing/chrome-extension_zoom.git
```

- 然后打开浏览器的`管理扩展程序`（开启开发者模式）
- 点击 `加载已解压的扩展程序`
- 然后选择从仓库克隆下来的文件夹
- 不出意外，应该就可以使用了
